# Favicon Allianza

Para adicionar o favicon do triângulo dourado com circuitos:

1. Salve a imagem como `favicon.png` ou `favicon.ico` neste diretório
2. Ou coloque em `static/favicon.ico` (raiz do static)
3. O código já está configurado para usar `/static/favicon.ico` ou `/static/images/favicon.ico`

Formato recomendado:
- PNG: 32x32, 64x64, ou 128x128 pixels
- ICO: formato tradicional de favicon (múltiplos tamanhos)
